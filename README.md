# Snapchat-Phone-Checker

Python script to automate the retrieval of Snapchat account phone numbers.

## Prerequisites

To run the checker script, ensure you have the following installed:

- **Python:** Version 3.6 or higher.
- **Google Chrome:** Latest version recommended.
- **Git:** For cloning the repository.
- **Proxies:** A list of working proxies in the format `IP:PORT:USER:PASS`.

## Installation

1. **Clone the Repository:**

    ```sh
    git clone https://github.com/Student-FastDev/Snapchat-Phone-Checker
    cd Snapchat-Phone-Checker
    ```

2. **Install Required Packages:**

    Install the necessary Python packages using `pip`:

    ```sh
    pip install -r requirements.txt
    ```

## Usage

1. **Run the Python Script:**

    Execute the script from the command line:

    ```bash
    python checker.py
    ```

2. **Configure Settings:**

    Prepare your input files to ensure the script runs properly:

    - **combo.txt:** Populate this file with your Snapchat account credentials in the format `username:password`, one per line.
    - **proxy.txt:** Edit this file to include your proxies in the format `IP:PORT:USER:PASS`, one per line.
    - **settings.json:** If the script generates a settings file upon the first run, review and adjust any configurable parameters as needed.

## Notes

- **Proxy Support & Reliability:** Utilizing high-quality and reliable proxies is crucial to maintain anonymity, prevent IP blocking, and ensure smooth operation.
